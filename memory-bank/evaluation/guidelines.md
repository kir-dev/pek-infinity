# Guidelines

Guidelines define the criteria for awarding points.

## Structure
-   **Description**: What the activity is.
-   **Point Range**: Min/Max points allowed.
-   **Approval**: Who can approve this activity (e.g., Project Leader).

## Usage
Guidelines are displayed to members so they know how to earn points.
